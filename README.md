# 투두리스트 앱
## 소개
이 프로젝트는 React와 Vite를 기반으로 구축된 현대적인 투두리스트 애플리케이션입니다.
## 데모
[🔗 라이브 데모 보기](https://todo-kohl-delta.vercel.app)
## 주요 기능
- ✅ 할 일 생성, 수정, 삭제 기능
- 🔄 실시간 상태 업데이트
- 📱 반응형 디자인
- 🎨 직관적인 사용자 인터페이스

## 기술 스택
- **프레임워크**: React 18.3.1
- **빌드 도구**: Vite 5.4.10
- **라우팅**: React Router DOM 7.0.1
- **스타일링**: 모던 CSS
- **아이콘**: Font Awesome, Lucide React
- **배포**: Vercel

## 개발 환경 설정
``` bash
# 프로젝트 클론
git clone [repository-url]

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
